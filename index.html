Frequently Asked Questions

What version is Nodenium on?

We use the latest version of Java edition Fabric 1.19


How do I join?

Click the 'Apply' button above to join the applications discord. Use the instructions in the #info channel.
You can message VyruZ#7158, Camo#0477, or Nate!#5922 if you have any questions. We're always happy to help!

How long does a season last?

There is no set duration for a season! We will keep going with a season as long as we are all still having fun. The last season was almost 10 months!

Where can I find a world download?

World files are available on the 'World Archives' tab in the Server Resources page

What are the rules?

- No getting items through exploits or unfair advantages (TNT duping is okay)
- No harming other players' progress (Stealing, Griefing, etc.)
- Be an active and responsible member of the community
These rules are common sense and up for interpretation, if you are uncertain about anything feel free to ask around!


How do I know if Nodenium is a good fit?

Nodenium is a community of talented, long-term, and dedicated, building-oriented players. If you're looking for a server that will truly last and allow you to fully express your building potential; then Nodenium is what you're looking for!

What are the requirements to join?

You must:
-Have a non-cracked version of the latest version of Minecraft Java
-Have a discord account
-Be a skilled builder
-Have a passion for creating something bigger than yourself
-And be a good person!

If you are unsure about anything feel free to join the discord and ask a team member! We're always happy to help!
